## 📂 Repository Contents

This repository contains five shell scripts developed as part of the Open Source Software assignment:

### 1. script1_system_identity.sh

This script displays detailed system information such as the hostname, current user, Linux distribution, kernel version, uptime, and current date. It uses commands like `uname`, `whoami`, and `uptime`.

### 2. script2_package_inspector.sh

This script checks whether Python (or any given package) is installed on the system. It also displays important details such as version, license, and description using package management commands.

### 3. script3_disk_auditor.sh

This script analyzes important system directories and reports their disk usage along with file permissions and ownership. It helps in understanding system storage and security.

### 4. script4_log_analyzer.sh

This script reads system log files and counts occurrences of keywords such as "error" or "warning". It helps in basic log monitoring and troubleshooting.

### 5. script5_manifesto.sh

This script interacts with the user to generate a simple open-source manifesto. It takes user input and creates a text file describing the importance of open-source software.
